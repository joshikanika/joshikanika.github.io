About



My journey started 5 years ago as a Software Engineer doing minor code enhancements and supporting web applications for a service-based company. The experience was nourishing for a newcomer trying to find her place in the IT world.


After two years I found myself working at a product based company that opened a whole bunch of opportunities to learn and grow. I am currently working as Technical Lead of level two support team that resolves escalated issues reported in web applications.


To fulfil my needs of continuous learning and development, I've started acquiring knowledge of MEAN stack to be able to build dynamic and modern websites.

Work

  • July 2012 - Jan 2015

    Software Engineer

    My responsibilities included requirement analysis with onsite team members, brainstorming the MOMs, enhancement of web applications, migration of applications from ASP to ASP.NET, handling incidents, change requests and problem tickets.

  • FEB 2015 - Sept 2016

    Application Specialist

    My role demanded resolving web application issues escalated by first level support team for Enterprise web applications and Microsoft Outlook Add-in, communicating with high maintenance global clients over emails, calls and video conferences for investigation and troubleshooting of their issues.

  • Oct 2016 - PRESENT

    Technical Lead

    My work includes deploying SaaS and hosted applications in multi-tenancy environments, hosting web applications in Microsoft Azure environment, suggesting process improvements, building internal applications for automating tasks to save time and efforts, managing projects for upgrading multiple clients applications, knowledge of Identity and Access Management, Single Sign-On functionality for applications hosted on cloud.
